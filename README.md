## 어글리마켓
 # 못난이 농산품 판매 사이트.

수확량의 1/3은 못난이 농산물로 분류되어 헐값에 처분되거나 폐기됩니다.

모양이 못나더라도 제 가치를 알려 불필요한 낭비를 줄이고 농가 소득 증진에 기여합니다.

***
### 페이지, 기능

  * 로그인, 가입하기 페이지
  
  * 상품 페이지
    * 전체,과일,채소 카테고리  
    
  * 장바구니 페이지
    * 카카오페이  
    
  * 레시피 게시판 
    * 게시판 등록, 수정, 삭제  
    * 댓글, 수정, 삭제 기능추가
